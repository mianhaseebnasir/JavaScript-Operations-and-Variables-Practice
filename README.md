<h1>JavaScript Operations and Variables Practice :</h1>

This project is a simple beginner-level JavaScript practice application.<br>
It demonstrates how basic arithmetic operations work in JavaScript and how variables can be used within a web page.<br>

The project uses HTML to create navigation links for different mathematical operations and JavaScript to implement logic related to variables.<br>

<h1>🚀 Features :</h1>
<li>Basic arithmetic operations</li>
<li>Beginner-friendly JavaScript examples</li>
<li>Simple HTML navigation interface</li>
<li>JavaScript variables implementation</li>
<li>Clean and easy-to-understand structure</li>

<h1>🧮 Operations Included :</h1>
This project includes the following mathematical operations:<br>
<li>Addition</li>
<li>Subtraction</li>
<li>Division</li>
<li>Multiplication</li>
<li>Modulus</li>
<li>Power</li>
<li>Percentage Calculator</li>
Each operation is available through a separate HTML page that can be accessed from the main page.

<h1>🛠️ Technologies Used :</h1>
<li>HTML5</li>
<li>JavaScript (Vanilla JS)</li>
No external libraries or frameworks are used in this project.

<h1>📖 How to Use :</h1>
<li>Download or clone the repository.</li>
<li>Open the project folder.</li>
<li>Open index.html in your web browser.</li>
<li>Click on any operation link to explore the functionality.</li>

<h1>🎯 Purpose of the Project :</h1>
The main purpose of this project is to help beginners understand:
<li>Basic JavaScript operations</li>
<li>How variables work in JavaScript</li>
<li>How HTML and JavaScript interact with each other</li>
This project is useful for students who are starting their journey in web development.
